Soal Latihan Pertemuan 10

  1. Apa gunanya IFRAME?
    Jawab : Untuk meng-embed dokumen lain atau menampilkan konten dari web lain kedalaman halam web yang sedang aktif , seperti video atau media interaktif lainnya.

  2. Apa bedanya FRAME dan IFRAME?
    Jawab :
      Frame
    • Tidak disarankan, karena sudah tidak sesuai dengan standard HTML saat ini
    • Frame digunakan untuk membagi satu halaman web menjadi beberapa bagian yang dapat menampilkan konten dari URL yang berbeda.
    • Frame dikelola di dalam elemen <frame> yang mengatur seberapa besar masing-masing frame dan di mana mereka harus muncul di halaman.

      • IFrame
    • Disarankan, Iframe adalah cara modern untuk menyisipkan konten eksternal ke dalam halaman web dan sesuai dengan standar HTML terkini.
    • Untuk membuat iframe, menggunakan elemen <iframe></iframe>yang dapat ditempatkan di mana saja di dalam halaman HTML.
    • Memiliki lebih banyak kendali atas iframe melalui atribut seperti src, width, height, dan sandbox.

  3. Tuliskan bentuk umum perintah IFRAME secara lengkap?
    Jawab : <iframesrc="" width="" height="" title="" ></iframe>
